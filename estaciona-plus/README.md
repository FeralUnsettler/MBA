# Estaciona+ Dashboard BI

Dashboard interativo para monitoramento do Estaciona+, plataforma para otimização de vagas de estacionamento.

## Como rodar localmente

### Pré-requisitos

- Python 3.8+
- pip

### Passos



Acesse o dashboard em: http://localhost:8501

---

## Rodando com Docker
